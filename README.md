# Verifast

## How do I install these formulae?

`brew install nyuichi/verifast/<formula>`

Or `brew tap nyuichi/verifast` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "nyuichi/verifast"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
